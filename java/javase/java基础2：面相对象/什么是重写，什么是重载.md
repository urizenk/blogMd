解答方向：多态与重写，方法签名与重载

1.重写就是子类将父类中的同名方法重新写入自己的实现方式，为了体现出面向对象的多态，将子类对象赋值给父类变量时，父类中的同名方法会自动的替换为子类中的方法，这样就可以在运行时同一个类表现出不同的行为方式。

2.重载就是同一个类中，参数列表不同的方法可以共用一个方法名而不冲突，只要不是一摸一样的参数列表就行，起到了对方法名的复用。
