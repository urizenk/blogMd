### 9.IN运算符

```mysql
select `value` in ('value1','value2','value3'...)
```

如果值在列表中，就返回1，不在就返回0，如果给定的值为null，返回null，如果列表中有null，并且没有命中，返回null。

### 11.LIKE运算符

模糊匹配，满足就返回1，不满足返回0，如果给定值与匹配条件为null，返回null。

(%):匹配0个或者多个字符。

(_):匹配一个字符。
